ACT2 --> Visor de Models AR



L’objectiu de la pràctica és preparar una primera petita aplicació com a demostració de coneixements mínims de Vuforia i Unity.  L’aplicació a desenvolupar ha de permetre la visualització d’un model projectat en realitat augmentada utilitzant el Software de Unity i Vuforia, a més d’un menú/slider/cojunt de botons que permetin seleccionar diferents models per tal de visualitzar-los sobre un Target de realitat augmentada fent servir un dispositiu mòbil amb l’aplicació instal·lada.
